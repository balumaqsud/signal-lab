# /run-prd

Execute a PRD implementation task end-to-end in this repository.

## Inputs

- PRD identifier/title
- Scope (backend/frontend/infra)
- Acceptance criteria

## Command behavior

1. Parse requirements into concrete implementation tasks.
2. Implement backend/frontend/infra changes as needed.
3. Apply project rules and relevant project skills.
4. Validate with lint/build/runtime checks.
5. Summarize completed criteria, remaining gaps, and next checks.
